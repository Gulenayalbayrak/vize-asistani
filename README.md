# Avrupa Vize ve Seyahat Portalı ✈️

Bu proje, vize süreçlerini dijitalleştirmek ve maliyet analizi yapmak için geliştirilmiş bir **RAG (Retrieval-Augmented Generation)** tabanlı yapay zeka asistanıdır. Microsoft Foundry Local altyapısını kullanarak verileri yerel olarak işler ve internet bağımlılığı olmaksızın cevap üretir.

## 🚀 Özellikler
* **Doküman Takibi:** Ülkeye özel gerekli evrak listesi.
* **Maliyet Analizi:** Vize harcı, sigorta ve servis ücretlerinin hesaplanması.
* **İstatistiksel Grafik:** Ülkelerin vize onay oranlarının görselleştirilmesi.
* **Çoklu Dil Desteği:** Türkçe ve İngilizce arayüz seçenekleri.

## 🛠️ Kullanılan Teknolojiler
* **Python**
* **Streamlit** (Arayüz)
* **Foundry Local SDK** (Yerel Yapay Zeka Runtime)
* **SQLite** (Vektör veritabanı)
* **Pandas** (Veri yönetimi)
